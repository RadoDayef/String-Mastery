# Strings Mastery

## What's Inside?

In this project, we cover:

1.  **String Properties:**
    *   `isEmpty`: Returns true if the string is empty.
    *   `isNotEmpty`: Returns true if the string is not empty.
    *   `length`: Returns the number of characters in the string.

2.  **String Methods:**
    *   `trim()`: Removes all whitespaces.
    *   `trimLeft()`: Removes leading whitespace.
    *   `trimRight()`: Removes trailing whitespace.
    *   `toLowerCase()`: Converts the string to lower case.
    *   `toUpperCase()`: Converts the string to upper case.
    *   `endsWith()`: Whether the string ends with a pattern.
    *   `startsWith()`: Whether the string starts with a pattern.
    *   `replaceAll()`: Replaces all substrings that match a pattern.
    *   `contains()`: Whether the string contains a match of a pattern.
